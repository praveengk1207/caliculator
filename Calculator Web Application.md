# Calculator Web Application

This is a simple web-based calculator application that allows users to perform basic arithmetic operations. You can use this calculator to add, subtract, multiply, divide, and more.

## Features

- Addition (+)
- Subtraction (-)
- Multiplication (\*)
- Division (/)
- Percentage (%)
- Clear All (AC)
- Delete Entry (DE)
- Decimal Point (.)

## Getting Started

To run this calculator application locally or integrate it into your website, follow these steps:

### Prerequisites

- Web browser (e.g., Chrome, Firefox, Safari)

## Usage

- To perform calculations, click the numeric buttons (0-9), operators (+, -, \*, /, %), and special functions (AC, DE, =).

- AC (All Clear): Clears the display.

- DE (Delete Entry): Removes the last character from the display.

- = (Equals): Evaluates the expression and displays the result.

- % (Percentage): Converts the current number to a percentage.

- Example: To calculate 5 + 3, enter `5 + 3 =`.

## Customization

You can customize the calculator's appearance and behavior by modifying the `style.css` file.
